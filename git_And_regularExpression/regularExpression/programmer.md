# 正则表达式语法

1. 元字符

    |元字符|描述|
    |-|-|
    |.|匹配除换行符以外的所有字符|
    |[ ]|匹配方括号中包含的任意字符|
    |[^ ]|不匹配方括号中出现的字符|
    |*|匹配前面的表达式0次或多次|
    |+|匹配前面的表达式1次或多次|
    |?|1.匹配前面的表达式0次或1次<br>2.非贪婪模式限定符|
    |{n,m}|匹配前面的表达式n-m次|
    |()|括号|
    |\||或|
    |\\|转义字符|
    |^|匹配行的开始|
    |$|匹配行的结束|

2. 简写字符集


    |简写|描述|
    |-|-|
    |\d|匹配数字|
    |\D|匹配非数字|
    |\s|匹配空白字符|
    |\S|匹配非空白字符|
    |\w|匹配字母数字下划线|
    |\W|匹配非字母数字下划线|
    |\b|匹配单词边界|
    |\B|匹配非单词边界|
    |\t|匹配制表符|
    |\n|匹配换行符|
    |\r|匹配回车符|
    |\f|匹配换页符|
    |\v|匹配垂直制表符|
    |\0|匹配空字符|
