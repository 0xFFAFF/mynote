# Latex语法

---

1. 自动匹配内容的大花括号

   $\left\{   \right\}$

2. 自动匹配内容的中括号

    $\left[   \right]$

3. 向上取整

    $\left \lfloor \right\rfloor$

4. 向下取整

    $\left\lceil  \right\rceil$

5. 任意符号

    $\forall$

6. 存在符号

    $\exists$
7. 属于/不属于

   $\in$  /  $\notin$

8.向右的箭头
    $\rightarrow$
